Redux-Thunk side effects for Kea

* kea-thunk 3.0 works with kea 3.0+
* kea-thunk 1.0 works with kea 1.0+
* kea-thunk 0.3 works with kea 0.27+
* kea-thunk 0.2 works with kea 0.26
* kea-thunk 0.1 works with kea 0.25
